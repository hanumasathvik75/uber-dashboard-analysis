# uber-dashboard-analysis
Analyzed Uber ride booking data to identify demand trends, cancellation patterns, payment preferences, and vehicle performance. Data was cleaned using Python (Pandas) and visualized through a multi-page interactive dashboard built in Power BI.
Uber Ride Booking Data Analysis
Project Overview

This project analyzes Uber ride booking data to understand ride demand, cancellations, payment preferences, and vehicle type performance.
Data cleaning was performed using Python (Pandas), and insights were visualized through an interactive Power BI dashboard.

Tools Used

Python

Pandas

Power BI

Data Cleaning Steps

The dataset was cleaned using Python before building the dashboard.

Cleaning tasks performed:

Removed duplicate records

Handled missing values (NA / NaN)

Standardized booking status and payment methods

Corrected inconsistent values

Dropped unnecessary columns

Dashboard Features

A 5-page Power BI dashboard was created containing:

KPI cards for total bookings, completed rides, cancelled rides

Monthly booking trend analysis

Vehicle type performance comparison

Payment method distribution

Cancellation analysis

Driver and customer ratings overview

Key Insights

Total Demand

The dataset contains around 150,000 ride bookings including completed, cancelled, and incomplete rides.

Monthly Trends

Ride bookings fluctuate across months, showing varying demand patterns rather than a steady trend.

Cancellations

Most ride cancellations are initiated by drivers.

Payment Methods

UPI is the most commonly used payment method, while card and bank payments are the least used.

Vehicle Type Performance

Auto rides dominate the platform, showing the highest bookings and travel distance.

Ratings

Driver and customer ratings remain fairly consistent across vehicle types.
conclusion

This project demonstrates how data cleaning and visualization can be used to uncover patterns in ride bookings, cancellations, and payment behavior. The dashboard provides useful insights into customer demand and operational trends.
